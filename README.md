# Terraform-EKS
In this project I configured with terraform the following aws resources, that will deploy "Hello world" python rest API ( link bellow ):
1. VPC
2. Network subnets
3. Routing tables
4. Nat gateways
5. Ineternet gateways
6. EKS cluster
7. Load Balancers
